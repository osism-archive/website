Deployment framework for OpenStack and Ceph

The Open Source Infrastructure & Service Manager is a powerful deployment framework for OpenStack and Ceph as well as required services such as a RabbitMQ broker or a MariaDB Galera cluster.

OpenStack provides a proven and reliable foundation for platforms such as Gitlab CI, Kubernetes, Cloud Foundry or OpenShift.

We manage our Betacloud, an OpenStack public cloud that provides cost-effective compute, storage and network resources for use in CI/CD systems, complex development & test environments, virtual training environments and more, with OSISM.
