Deployment & lifecycle management framework for OpenStack & Ceph

The Open Source Infrastructure & Service Manager is a powerful deployment and lifecycle management framework for OpenStack and Ceph as well as required services such as a RabbitMQ broker or a MariaDB Galera cluster.

OpenStack provides a proven and reliable foundation for platforms such as Gitlab CI, Kubernetes, Cloud Foundry, OpenShift, or High Performance Computing (HPC).

OSISM has been successfully deployed on a variety of production environments for over 4 years for operations and lifecycle management. As the basis for pluscloud open from the German-based cloud service provider PlusServer in Cologne, OSISM is an integral part of one of the first GAIA-X hosters.

Sovereign Cloud Stack

OSISM is an integral part of the Sovereign Cloud Stack (SCS) project funded by the Spring Innovation Agency. It will thus play a major role in the GAIA-X project.

For German-speaking readers, iX 2020/12 contains an article with the title Wolken-Verbund written by Kurt Garloff. This article presents in detail the vision behind the Sovereign Cloud Stack and the current state of technical implementation based on OSISM.

Contact us for more information about Sovereign Cloud Stack and GAIA-X.

Enterprise Edition

OSISM is a 100% open source solution, built on open source building blocks, and it is developed openly and transparently. For use in enterprise productions, the use of the OSISM Enterprise Edition provided by 23 Technologies is recommended. The OSISM Enterprise Edition enables the use of LTS versions, managed services, professional support, vulnerability scanning, signed images, and more.

The OSISM Enterprise Edition will be fully SCS compliant and certified, as soon as a relevant certification is available. This enables the baseline for building future GAIA-X compliant cloud solutions like pluscloud open.

For German-speaking readers, iX 2021/4 contains an extensive article with the title Die Europa-Cloud kommt written by Martin Gerhard Loschwitz on the use of OSISM in pluscloud open.
